%{
#include <stdio.h>
#include <string.h>

char symtab[100][50];
int count = 0;

void add_to_table(char *word) {
    for(int i = 0; i < count; i++) {
        if(strcmp(symtab[i], word) == 0)
            return;   // avoid duplicates
    }
    strcpy(symtab[count++], word);
}
%}

# 🎯 **Assignment 3(b) Explanation (Ready to Speak)**

## 🧾 **1. Introduction**

This program implements a **lexical analyzer using LEX (Flex)** for identifying **parts of speech in a subset of English language with a symbol table** .

A lexical analyzer scans input and converts it into **tokens**.
In this program, tokens are parts of speech such as:

* Noun
* Pronoun
* Verb
* Adjective
* Adverb
* Preposition
* Conjunction
* Interjection

Additionally, this program also **stores the words in a symbol table**.

---

## 🎯 **2. Objective**

The objective is to:

* Read an English sentence
* Identify the part of speech of each word
* Store the words in a **symbol table**
* Avoid duplicates in the table

---

## ⚙️ **3. Working of the Program**

This program is written using **LEX**, which uses **pattern matching**.

It has 3 sections:

---

### 🔹 (1) Definition Section

```c
%{
#include <stdio.h>
#include <string.h>

char symtab[100][50];
int count = 0;
%}
```

👉 Here:

* `symtab` → stores words
* `count` → number of stored words

---

### 🔹 (2) Rules Section (Main Logic)

Example:

```lex
"he"|"she"|"we" {
    printf("Pronoun: %s\n", yytext);
    add_to_table(yytext);
}
```

👉 What happens:

1. Word is matched
2. Category is printed
3. Word is stored in symbol table

---

### 🔹 Symbol Table Function

```c
void add_to_table(char *word) {
    for(int i = 0; i < count; i++) {
        if(strcmp(symtab[i], word) == 0)
            return;
    }
    strcpy(symtab[count++], word);
}
```

👉 This function:

* Checks if word already exists
* If not → adds it
* Prevents duplicates

---

### 🔹 (3) User Section

```c
int main() {
    yylex();

    for(int i = 0; i < count; i++) {
        printf("%s\n", symtab[i]);
    }
}
```

👉 `yylex()` scans input
👉 After scanning → symbol table is printed

---

## 🧠 **4. Important Concepts**

### 🔸 What is Symbol Table?

* A data structure used to store tokens
* Here it stores **unique words**

---

### 🔸 What is `yytext`?

* Stores the matched word

---

### 🔸 Why avoid duplicates?

* To reduce redundancy
* Efficient storage
