## 🧾 **1. Introduction**

This program implements a **lexical analyzer using LEX (Flex)** for a **subset of the C programming language with a symbol table** .

A lexical analyzer scans the input program and breaks it into **tokens** such as:

* Keywords
* Identifiers
* Numbers
* Operators
* Punctuation symbols

In this program, along with identifying tokens, we also **store identifiers in a symbol table**.

---

## 🎯 **2. Objective**

The objective is to:

* Read C language input
* Identify different tokens
* Store **identifiers in a symbol table**
* Avoid duplicate entries
